Um pouco mais da nossa história...


Esse projeto foi idealizado com a participação de seis estudantes em um programa desenvolvido pelo Facebook, em parceria com a empresa Digital House, chamado "Front End Coding Facebook". O intuito do programa foi de capacitar jovens para se tornarem desenvolvedores Front End, um mercado em contante desenvolvimento, que necessitará cada vez mais de pessoas capacitadas para atenderem a demanda do mercado. Para realizar o projeto, os jovens desenvolvedores deveriam encontrar algum tema de comum interesse, e procurar uma API que retornariam dados para serem consumidos no site desenvolvido, com intuito de colocar em prática todo o aprendizado desenvolvido no decorrer do curso.
